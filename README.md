# 📖 Vocabulary Learning App

## 📖 Overview
This is a **Vocabulary Learning App** where users can explore words by lesson levels, search meanings, and even listen to pronunciation using text-to-speech.  
The app focuses on **API integration, dynamic UI rendering, and real user interaction**, making it feel like a complete product rather than just a static webpage.

## 🛠️ Main Technologies Used
- JavaScript (Vanilla JS)
- REST API
- Tailwind CSS
- DaisyUI
- Web Speech API (Text-to-Speech)

---

## ✨ Core Features
- Dynamic lesson levels loaded from API  
- Words displayed based on selected level  
- Search functionality using `filter()`  
- Text-to-speech pronunciation feature  
- Responsive navbar & banner design  
- Loading spinner for async operations  
- Modal for word details (DaisyUI)  
- Active lesson highlighting  
- Handles empty states & null data (better UX)

---

## 📦 Dependencies
- tailwindcss  
- daisyui  
- font-awesome / icons library  
- (No framework – pure JavaScript project)
